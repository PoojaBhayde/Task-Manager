# Task Manager API (Node.js + MySQL)

A RESTful Task Manager API built with Node.js, Express, and MySQL that allows users to manage tasks with authentication, status tracking, and relational data handling.

---

## 🚀 Features
- User Authentication (JWT)
- CRUD operations for tasks
- Task pagination
- MySQL relational database
- Secure password hashing
